## 🤝 Contributing

Contributions are **very welcome** and what make this list great! 🎉

Read [**CONTRIBUTING.md**](CONTRIBUTING.md) to get started. In short:

1. **Fork** this repository
2. **Add** your project under the correct category in `README.md`
3. **Format:** `| [**Name**](github-url) | One-line description | [![Stars](badge-url)](github-url) |`
4. **Verify:** Project must be open source, actively maintained, and on GitHub/GitLab
5. **Open a PR** with title: `Add [Project Name] to [Category]`

> **Looking for something to add?** Check [open issues](../../issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22) tagged `good first issue`!

### Contribution Guidelines

| ✅ Accepted | ❌ Not accepted |
|------------|----------------|
| Open source with a clear license | Closed source or proprietary tools |
| Actively maintained (commits in last 12 months) | Abandoned projects (2+ years no activity) |
| Solves a real developer or user problem | Pure clones with no value-add |
| Has a GitHub/GitLab repository | No source code available |
